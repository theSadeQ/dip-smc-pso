## Responsive Design Strategy

### Desktop (1024px+)
```
Full Mermaid diagram width
No scaling
Standard font sizes
Hover tooltips enabled
```

### Tablet (768-1023px)
```
75-90% width
Slight font reduction
Touch-friendly spacing
Simplified hover (optional)
```

### Mobile (320-767px)
```
100% width with padding
60-80% scale or text-only fallback
Collapsible wrapper for large diagrams
Larger touch targets
Readable font (12px minimum)
```
