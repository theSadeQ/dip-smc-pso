Double Inverted Pendulum Control

Advanced sliding mode control with PSO optimization for double inverted pendulum systems.

Key Features

  • Multiple SMC controllers (Classical, STA, Adaptive, Hybrid)
  • PSO parameter optimization
  • Real-time visualization
  • Hardware-in-the-loop testing

Getting Started

Install dependencies and run your first simulation in under 5 minutes.

pip install -r requirements.txt
python simulate.py --ctrl classical_smc --plot
