# Ultrathink Session Prompts - Systematic Audit Execution

**Purpose**: Pre-designed session prompts for thorough, manageable audits using sequential-thinking.
**Date**: November 9, 2025

---

## How to Use These Prompts

1. **Copy** the full prompt (including all sections)
2. **Trigger** sequential-thinking MCP server by starting your message with the prompt
3. **Follow** the systematic analysis workflow
4. **Validate** using the included verification steps
5. **Deliver** all specified outputs

**Expected Duration**: 2-8 hours per session (as specified in each prompt)

## Standard Workflow

1. **Choose Prompt** - Select based on audit scope (quick/medium/comprehensive)
2. **Customize** - Fill in TARGET placeholders (filename, controller, category)
3. **Trigger Sequential Thinking** - Paste full prompt to start systematic analysis
4. **Follow Tasks** - Complete each analysis task in order
5. **Validate** - Execute all validation requirements
6. **Deliver** - Produce all specified deliverables
7. **Verify Success** - Check all success criteria

---

## Tips for Success

1. **Don't Skip Validation** - Validation requirements are non-negotiable
2. **Document Everything** - Save all intermediate results (scripts, data, notes)
3. **Use Automation** - Write scripts for data collection (don't do manually)
4. **Verify Manually** - Always spot-check automated results
5. **Be Specific** - Include line numbers, file paths, exact errors
6. **Estimate Effort** - All recommendations need time estimates
7. **Answer "So What?"** - Provide context and interpretation
