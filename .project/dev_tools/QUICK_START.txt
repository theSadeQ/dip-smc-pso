================================================================================
    CLAUDE CODE MULTI-ACCOUNT OAUTH - QUICK START
================================================================================

PROBLEM: OAuth tokens expire after 8 hours in Claude Code 2.0.36
SOLUTION: Browser profiles with automated OAuth (95% automation)

================================================================================
    INITIAL SETUP (ONE-TIME - ~65 MINUTES TOTAL)
================================================================================

Step 1: Create Browser Profiles (5 minutes)
--------------------------------------------
powershell> .\Setup-BrowserProfiles.ps1 -InitialSetup

- Choose browser: Chrome (recommended), Edge, or Firefox
- Choose accounts: All 32, specific, or only expired
- Profiles created automatically

Step 2: Authenticate Profiles (60 minutes)
-------------------------------------------
powershell> .\Authenticate-AllProfiles.ps1

For each account:
  1. Browser opens with dedicated profile
  2. Log in to claude.com
  3. Check "Stay logged in"
  4. Close browser
  5. Press Enter to continue

Step 3: Verify Setup (1 minute)
--------------------------------
powershell> .\Setup-BrowserProfiles.ps1 -ValidateProfiles

Should show all 32 accounts as [OK]

================================================================================
    DAILY USAGE
================================================================================

Check Token Status
------------------
powershell> .\Monitor-ClaudeTokens.ps1

Shows:
  [OK] Account 12: 7.8 hours left
  [WARN] Account 5: 1.5 hours left
  [ERROR] Account 3: EXPIRED

Refresh Single Account
----------------------
powershell> .\Auto-RefreshToken-Browser.ps1 -AccountNum 12

Process (10 seconds):
  1. Browser opens with profile 12
  2. Open new terminal -> run: claude
  3. Type: /login
  4. Browser auto-completes OAuth
  5. Token refreshed!
  6. Browser closes

Refresh All Expiring
--------------------
powershell> .\Auto-RefreshToken-Browser.ps1 -RefreshAll

Batch refreshes all accounts with < 2 hours left

Continuous Monitoring
---------------------
powershell> .\Auto-RefreshToken-Browser.ps1 -Watch

Checks every 30 minutes, auto-refreshes when needed

================================================================================
    TROUBLESHOOTING
================================================================================

Error: "Configuration not found"
---------------------------------
Fix: .\Setup-BrowserProfiles.ps1 -InitialSetup

Error: "Profile not found for account N"
----------------------------------------
Fix: .\Setup-BrowserProfiles.ps1 -AccountNum N
     .\Open-ProfileForAuth.ps1 -AccountNum N

Error: Browser keeps asking for login
--------------------------------------
Fix: .\Open-ProfileForAuth.ps1 -AccountNum N
     Log in again, check "Stay logged in"

Error: Multiple browser windows accumulate
-------------------------------------------
Fix: Stop-Process -Name chrome -Force
     Stop-Process -Name msedge -Force
