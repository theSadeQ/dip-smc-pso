# Orchestration

Agent profiles and MCP coordination for multi-agent tasks.

## Directory Structure

- `agents/` - Agent profiles (8 specialized agents for orchestration)
- `mcp/` - MCP configuration (usage guide, custom MCP servers)
- `commands/` - Slash commands (9 command templates)
- `docs/` - Orchestration documentation

## Purpose

Central hub for coordinating multi-agent workflows and MCP server integrations.
