# Development

Development tools and automation for code quality and research.

## Directory Structure

- `analysis/` - Code analysis tools (quality audit, coverage audit, performance analyzer)
- `automation/` - Build & CI scripts (git hooks, task scheduler)
- `research/` - Research pipeline (citation database, batch processing)
- `quality/` - Quality improvement scripts
- `docs/` - Automation guides

## Purpose

Tools for maintaining code quality, automating workflows, and conducting research.
