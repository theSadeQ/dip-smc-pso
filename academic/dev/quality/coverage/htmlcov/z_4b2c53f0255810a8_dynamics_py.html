279 """ 

280 Single-step Euler integration (for batch simulation compatibility).

282 Args:

284 control_input: Control input (scalar or 1D array) 

285 dt: Time step

287 Returns: 

288 Next state using forward Euler integration

290 Note: 

291 This method provides compatibility with batch simulation frameworks

293 with your own integrator. 

294 """ 

295 # Normalize control input to array format 

296 if np.isscalar(control_input): 

297 control_input = np.array([control_input]) 

298 elif isinstance(control_input, np.ndarray) and control_input.ndim == 0:

305 return state
