## How to Use with NotebookLM

### Quick Start (15 minutes)

1. Go to https://notebooklm.google.com
2. Create a new notebook
3. Upload `pdf/E001_project_overview_and_introduction.pdf`
4. Click "Generate" to create audio overview
5. Download the generated podcast

### Batch Processing (2-3 hours)

1. Create 5-6 separate NotebookLM notebooks
2. Upload 5 episodes per notebook (max sources per notebook)
3. Generate audio for each notebook
4. Download all podcasts
5. Combine into a complete learning series
