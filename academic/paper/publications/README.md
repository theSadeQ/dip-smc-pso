## Organization

Each paper has its own subdirectory with the following structure:

- **Root:** Final PDF, final TEX, bibliography, README (4-5 files only)
- **source/:** Working versions (markdown, HTML, alternative TEX)
- **submission/:** Cover letters, checklists, reviewer suggestions, manuals
- **reports/:** Progress reports, completion summaries
- **sections_source/:** Section-level sources with organized subdirectories
  - **scripts/:** Automation tools for section extraction, PDF generation, audits
  - **config/:** Configuration files for automation workflows
  - **guides/:** Usage documentation and quick-start guides
  - **audits/:** Quality assurance reports and audit summaries
- **archive/:** Historical versions, planning docs, test files, build artifacts
