\section{Troubleshooting}

\subsection{Can't Print Checklist?}
\begin{enumerate}
    \item Open PDF in browser
    \item Use browser print (Ctrl+P)
    \item Export to different format if needed
    \item Try online PDF print services
\end{enumerate}

\subsection{VS Code Settings Not Working?}
\begin{enumerate}
    \item Verify file is at \texttt{.vscode/settings.json} in project root
    \item Check syntax (must be valid JSON)
    \item Restart VS Code
    \item Check for conflicting extensions
\end{enumerate}

\subsection{Commands Not Working?}
\begin{enumerate}
    \item Verify you're on Windows (use \texttt{python} not \texttt{python3})
    \item Check you're in project root: \texttt{D:\textbackslash Projects\textbackslash main\textbackslash}
    \item Ensure dependencies installed: \texttt{pip install -r requirements.txt}
    \item Check Python version: \texttt{python --version} (should be 3.9+)
\end{enumerate}

\subsection{Progress Tracker Too Detailed?}
\begin{enumerate}
    \item Skip tables/metrics if not relevant
    \item Focus on reflections and understanding ratings
    \item Customize sections as needed
    \item Use it as lightly or heavily as you prefer
\end{enumerate}
