# Educational Materials

> Learning resources for DIP-SMC-PSO framework, from beginners to framework architects.

**Purpose:** Provide structured learning paths separate from the technical reference docs.

---

## Available Resources

- **Layered Learning Roadmap** (main guide)
  - File: `.ai_workspace/edu/layered-learning-roadmap.md`
  - Coverage: All 358 Python files (~210K LOC)
  - 5 layers: Foundation -> Core -> Advanced -> Integration -> Mastery
  - Includes checkpoints and verification tests

- **Quick Layer Reference** (fast lookup)
  - File: `.ai_workspace/edu/quick-layer-reference.md`
  - Decision tree, file-to-layer lookup, short checks

- **Visual Learning Aids** (ASCII diagrams)
  - File: `.ai_workspace/edu/visual-learning-aids.md`

---

## Recommended Starting Points

- New to the project -> Start at Layer 1 in the roadmap
- Control theory background -> Start at Layer 2, then backfill Layer 1
- Software engineer -> Focus Layers 1, 2, and 4
- Control engineer -> Focus Layers 1, 2, and 3
- Deadline project -> Use project-driven strategy in quick reference
